# Бюджетный помощник

Консольная программа для учета личных трат за один месяц. Пользователь записывает трату в формате день, сумма, категория, а программа считает суммы за любой период с дня A по день B за O(1), ищет самый дорогой день, строит отчет по категориям и показывает траты по возрастанию суммы.

## Что делает программа

- Добавляет трату с днем, суммой и категорией
- Отменяет последнюю добавленную трату
- Считает сумму трат за период с дня A по день B за O(1)
- Находит день, в который потрачено больше всего
- Показывает итог за месяц и средний расход за день с тратами
- Сортирует категории по сумме трат
- Выводит все траты по возрастанию суммы
- Находит самую дорогую и самую дешевую трату
- Показывает траты по дням

## Файлы проекта

- `structures.py` содержит стек `Stack`, узел `Node` и дерево `BudgetTree`
- `budget.py` содержит класс `Expense` для одной траты и класс `Budget`, который ведет учет
- `reports.py` содержит сортировку вставками и подсчет трат по категориям
- `main.py` содержит меню и главный цикл

## Как все связано

Список `expenses` хранит все траты по порядку. Массив `per_day` помнит сумму трат за каждый день. Массив `prefix` строится из `per_day` по запросу и отвечает на запрос периода за одно вычитание. Дерево `tree` хранит те же траты, упорядоченные по сумме. Стек `history` помнит порядок добавления для отмены. При отмене трата убирается из списка, из дневной суммы и из дерева, а префиксные суммы помечаются как устаревшие.

## Структуры данных и алгоритмы

- Префиксные суммы дают ответ на запрос суммы за период за O(1)
- Стек хранит историю добавленных трат для отмены последней
- Бинарное дерево поиска хранит траты по сумме, дает отсортированный список через симметричный обход и быстро находит минимум и максимум, а при отмене узел удаляется из дерева
- Сортировка вставками упорядочивает категории по сумме трат
- Линейный поиск находит день с максимальным расходом
